📋 DEFINITION OF DONE (để S5 dùng làm checklist kiểm thử thực tế):
□ GET /api/admin/dashboard trả 200 với 6 trường: totalUsers, newUsersThisWeek,
  newUsersThisMonth, totalExamSessions, examPassRate (số thực), onlineNow
□ GET /api/admin/users trả 200 với phân trang đúng (totalPages >= 1 dù không có kết quả)
□ GET /api/admin/users?search=<tên> trả đúng kết quả (case-insensitive)
□ GET /api/admin/users?isBlocked=true chỉ trả user bị khoá
□ GET /api/admin/users?role=ADMIN chỉ trả user có role=ADMIN
□ GET /api/admin/users/:id trả 200 với user.stats.totalPracticeSessions, recentExams (≤5)
□ GET /api/admin/users/:id với ID không tồn tại → 404 ADMIN_USER_NOT_FOUND
□ PATCH /api/admin/users/:id/block { isBlocked: true } → user đó nhận 403 USER_BLOCKED
  khi gọi bất kỳ API nào cần xác thực
□ PATCH /api/admin/users/:id/block { isBlocked: false } → user đó có thể dùng app lại
□ POST /api/admin/users/:id/reset-password với user có email → trả resetLink
□ POST /api/admin/users/:id/reset-password với user không có email → 400 ADMIN_USER_NO_EMAIL
□ PATCH /api/admin/users/:id/role { role: "ADMIN" } → user.role = "ADMIN"
□ PATCH /api/admin/users/:id/role { role: "INVALID" } → 400 ADMIN_INVALID_ROLE
□ DELETE /api/admin/users/:id → 200 + Firebase account bị xóa + DB record bị xóa
□ Tất cả 7 endpoint → 401 ADMIN_UNAUTHORIZED khi không gửi X-Admin-Secret
□ Tab 📊 Dashboard trên frontend hiển thị 6 thẻ thống kê có giá trị thực
□ Tab 👥 Người dùng hiển thị danh sách, tìm kiếm, lọc, phân trang hoạt động đúng
□ Modal chi tiết user mở khi click vào 1 dòng, hiển thị stats + recentExams
□ Nút Khoá/Mở khoá, Reset mật khẩu, Đổi role, Xóa (có confirm dialog) hoạt động đúng
□ Unit test: 44/44 PASS (npm run test trong thư mục backend)

🌐 API MỚI:
- GET /api/admin/dashboard
- GET /api/admin/users (query: search, role, isBlocked, page, limit)
- GET /api/admin/users/:id
- PATCH /api/admin/users/:id/block (body: { isBlocked: boolean })
- POST /api/admin/users/:id/reset-password
- PATCH /api/admin/users/:id/role (body: { role: "STUDENT"|"ADMIN" })
- DELETE /api/admin/users/:id

🗄️ DATABASE:
- Bảng users: thêm isBlocked (Boolean default false), role (String default "STUDENT")
- Index: users_isBlocked_idx, users_role_idx
