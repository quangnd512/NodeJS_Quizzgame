DB Schema cần thêm vào model User (backend/prisma/schema.prisma):
  isBlocked Boolean  @default(false)   // khoá tài khoản
  role      String   @default("STUDENT") // "STUDENT" | "ADMIN"

API Draft đầy đủ tại: docs/api/drafts/admin-user-management.yaml

Edge case quan trọng:
- verifyAppToken phải check isBlocked → trả HTTP 403 (không phải 401)
- Online users: mỗi lần verifyAppToken thành công → SET Redis key "online:{userId}" EX 300
  Count online = đếm keys matching "online:*" (hoặc dùng SCAN)
- Reset password: dùng getFirebaseAuth().generatePasswordResetLink(email)
  → trả link về cho admin (không gửi email tự động)
  → trả 400 nếu user không có email
- Xoá user: xoá Firebase trước (getFirebaseAuth().deleteUser(firebaseUid))
  rồi mới xoá DB — nếu DB lỗi thì đã mất Firebase user (chấp nhận được, ghi log)
- User đang làm bài thi mà bị khoá: bài thi hiện tại vẫn tiếp tục (không force-stop),
  chặn ở lần request tiếp theo
- Dashboard pass rate: ExamSession.status='COMPLETED' AND score >= 7.0
